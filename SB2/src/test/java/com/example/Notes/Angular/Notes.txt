	Install node js ,  VSCode software and Typescript
Step-2
	open angular project folder install angular cli
command 1 : npm install -g @angular/cli
-g means Global instalation 
command 2 : npm install -g typescript
command 3 : npm install, npm start,  npm update 
command 4:  check node version : node -v
command 5: check angular cli : npm --version
create a new Angular Application: ng new <project_name>


Run the project Command 
npm start or ng serve

we change port number also command 
			1. ng serve -p 5000 
			2. ng serve -o
	please enter two commands

ng g c header
